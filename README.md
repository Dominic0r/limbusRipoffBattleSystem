My attempt at recreating Limbus Company's battle system (not really 100% accurate though). I really liked the system and saw potential in status effects if they're combined with complex scription

This program contains a very flexible status effects system that allows users with the know-how to script custom status effects using java.

Instructions on how to run:
Prerequisites: You must have the latest Java Development Kit (JDK) installed

Step 1: Download the zip file via the green <> Code button on the top right

Step 2: Unzip the files in any directory / folder

Step 3: Open a terminal in that directory / folder

Step 4: Compile the program by entering javac Main.java

Step 5: Run the program by entering java Main.java



Instructions on modding:

Prerequisites: Baseline knowledge on how to code in Java

There will be formats provided for each aspect
